# Deriv Accumulator AI Sniper Pro

A high-performance trading tool for Deriv's Accumulator contracts.

## Key Features
- **AI Sniper Execution**: Ultra-low latency trade entry using WebSocket proposal streams.
- **Dynamic Risk Management**: Multi-layer exit engine analyzing price velocity, barrier proximity, and trend reversals.
- **Global Market Scanner**: Real-time monitoring of 10 volatility symbols with automated stability scoring.
- **ARIA Voice AI**: Interactive voice assistant for hands-free trading and market updates.
- **Pro-Grade Charts**: Integrated TradingView Lightweight Charts with custom barrier tracking.

## Technical Setup
- **Server**: Python HTTP Server (Port 5000).
- **Frontend**: Vanilla JS/HTML5/CSS3.
- **API**: Deriv WebSocket (App ID 129093).
- **Auth**: Secure OAuth2 session management.

- `index.html`: Unified application codebase.
